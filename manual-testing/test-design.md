# Prompt Name
Test Case Design Prompt

## Purpose
This prompt helps generate structured functional test cases for a web application or software feature, ensuring full coverage of normal, edge, and negative scenarios.

## Prompt

You are a senior QA engineer.

Analyze the following feature description and generate comprehensive functional test cases.

Focus on:
- positive scenarios
- negative scenarios
- edge cases
- expected results
- preconditions

Feature description:
[PLACE FEATURE DESCRIPTION HERE]

Output format:
| Test Case ID | Scenario | Steps | Expected Result |
