# Prompt Name
Exploratory Testing Prompt

## Purpose
This prompt guides QA engineers to perform structured exploratory testing, uncovering unexpected behaviors, usability issues, and edge cases that scripted tests may miss.

## Prompt

You are a senior QA engineer performing exploratory testing.

Analyze the following feature or module and generate a list of potential exploratory test scenarios.

Focus on:
- unexpected behaviors
- usability issues
- edge flows
- risk-prioritized testing

Feature description:
[PLACE FEATURE DESCRIPTION HERE]

Output format:
| Test Scenario ID | Scenario Description | Steps to Explore | Potential Risks |
