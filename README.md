# QA Prompt Library

This repository contains reusable prompts for QA engineers to improve testing workflows using AI tools.

## Structure

- manual-testing/
  - Test Case Design
  - Exploratory Testing
  - Bug Analysis

## How to Use

1. Copy a prompt
2. Paste into AI tool
3. Replace placeholders
4. Generate results
